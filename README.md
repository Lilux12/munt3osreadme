📖 About

munt3os is an experimental operating system project in its early stages of development. It aims to provide a solid, understandable foundation for building a more complex operating system in the future, taking advantage of Rust's safety and performance features.

✨ Features

Kernel & Memory

Memory Management: Physical (pmm) and Virtual (vmm) memory management.

Allocators: Heap, Slab, and Buddy allocators implemented.

Processes: Basic process management and scheduling.

User-space: Support for user-space applications (currently running in ring 0).

Hardware & Drivers

Interrupts: Core interrupt handling.

Basic Devices: Keyboard and Timer drivers.

Storage Controllers: Support for AHCI (SATA) storage devices.

USB Stack: Support for xHCI (USB 3.0) and OHCI (USB 1.1).

File Systems

Supported FS: tarfs, ext2.

💻 Supported Hardware

Keyboard: Generic USB Keyboard (USB HID Class, BOOT)

More devices planned (see https://www.google.com/search?q=TODO.md)

Building and Running

cd munt3os

# 2. Build the kernel
cargo build --release

# 3. Run in QEMU
# (Assuming you have a Makefile or runner script configured with Limine)
make run
